# Research Agent API Services

This document outlines all the available HTTP endpoints exposed by the FastAPI application in the Research Agent project.

## 1. POST `/research`
- **Description:** Triggers the autonomous LangGraph agent to research a given topic on the internet and write a comprehensive, cited Markdown report. *Note: This endpoint is synchronous and may block for up to a minute depending on the complexity of the research.* 
- **Content-Type:** `application/json`
- **Parameters (JSON Body):**
  - `topic` (String, required): The broad subject or specific question you want researched. Example: `"What are the top 3 biggest AI news stories from March 2026?"`
- **Response (JSON):**
  - `topic`: String (The original request topic)
  - `report_markdown`: String (The fully synthesized, formatted Markdown report generated by the autonomous agent)

---

>**Note:** FastAPI automatically provides interactive Swagger documentation for all these routes natively at `http://localhost:8003/docs` while the application is running via `make run-research` or `make run-docker-research`.
